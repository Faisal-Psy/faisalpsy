# Personal Website — Md Faisal Ahmed

This is a simple, responsive personal website template built for Md Faisal Ahmed.
Drop your CV `CV_MD-Faisal-Ahmed.pdf` into the same folder and update content as needed.

Quick start
1. Put these files in a directory:
   - index.html
   - assets/css/style.css
   - assets/js/main.js
   - assets/images/headshot-placeholder.png (replace with your photo)
   - CV_MD-Faisal-Ahmed.pdf (your CV file)

2. Open `index.html` in a browser to preview.

Deploy to GitHub Pages
1. Create a new GitHub repository and push these files to the `main` branch.
2. In the repository `Settings` → `Pages`, set Source to `main` branch `/ (root)`.
3. Save. Your site will be available at `https://<yourusername>.github.io/<repo>` shortly.
